Shrug Whitepaper

1. Project Overview

Shrug is a decentralized token project built on the Ethereum blockchain, designed to facilitate an innovative token purchase and reward mechanism integrated with the WoofSwap decentralized exchange. The Shrug token (symbol: SHRUG) is an ERC20 token that incentivizes participation in the memecoin ecosystem. Users can purchase whitelisted memecoins using a fixed amount of BONE tokens and receive SHRUG tokens as a reward. Shrug aims to promote engagement in the memecoin market while ensuring security and transparency.

Core Functionality: Users pay 0.01 BONE to purchase whitelisted memecoins via WoofSwap and receive 1,000,000 SHRUG tokens as a reward.

2. Tokenomics

The Shrug token (SHRUG) is the core asset of the project, governed by the following tokenomics:

  • Token Name: Shrug
  • Symbol: SHRUG
  • Decimals: 18
  • Maximum Supply: 1,000,000,000,000,000 SHRUG
  • Reward Mechanism: Each valid purchase yields 1,000,000 SHRUG, minted until the maximum supply is reached.
  • Distribution: Tokens are minted and distributed to users upon successful memecoin purchases, with no pre-minted supply.

The contract ensures that the total supply never exceeds the maximum limit, maintaining scarcity and fairness in distribution.

3. Core Mechanisms

3.1 Purchase Mechanism

Users can participate in the Shrug ecosystem by calling the buy function, which requires sending exactly 0.01 BONE to purchase a whitelisted memecoin. The process is as follows:

  • Input Validation: The contract verifies that the sent amount is 0.01 BONE and the target token is a whitelisted memecoin with a valid WoofSwap pair.
  • Swap Execution: The contract interacts with the WoofSwap router (0x96b16aBD53Bfd765F4CD118590C1d0be8B57DE24) to swap BONE for the target memecoin, applying a 95% slippage tolerance and a 300-second deadline extension.
  • Reward Distribution: Upon successful swap, the user receives 1,000,000 SHRUG tokens, provided the total supply does not exceed the maximum limit.

The purchase process is secured by the nonReentrant modifier to prevent reentrancy attacks and ensure transaction integrity.

3.2 Memecoin Whitelisting

Only the contract owner can manage the memecoin whitelist via the addMemecoins and removeMemecoin functions. This ensures that only verified memecoins with valid WoofSwap pairs are eligible for purchase, maintaining ecosystem quality.

3.3 Fund Management

The contract includes mechanisms for the owner to withdraw accumulated funds:

  • withdrawBone: Transfers ERC20 BONE tokens (0x839FdB6cc98342B428E074C1573ADF6D48CA3bFd) held by the contract to the owner.
  • emergencyWithdraw: Transfers native BONE (or other native tokens) held by the contract to the owner in case of emergencies.

Both functions are restricted to the owner and protected by the nonReentrant modifier for security.

4. Security Features

Shrug prioritizes security and reliability through the following measures:

  • Ownership Control: Only the contract owner can perform sensitive operations like whitelisting memecoins or withdrawing funds, enforced by the onlyOwner modifier.
  • Reentrancy Protection: The nonReentrant modifier prevents reentrancy attacks during fund withdrawals and token purchases.
  • Slippage Protection: A 95% slippage tolerance is applied during swaps to protect users from excessive price volatility.
  • Input Validation: The contract enforces strict checks on token addresses, amounts, and pair validity to prevent errors or exploits.

5. Integration with WoofSwap

Shrug leverages the WoofSwap decentralized exchange for its swap functionality. Key integrations include:

  • Router: 0x96b16aBD53Bfd765F4CD118590C1d0be8B57DE24 for executing token swaps.
  • Pair Factory: 0xB9fbdFA27B7ba8BB2d4bB4aB399e4c55F0F7F83a for verifying valid trading pairs.
  • BONE Token: 0x839FdB6cc98342B428E074C1573ADF6D48CA3bFd as the primary input token for purchases.

This integration ensures seamless and trustless token swaps, aligning with Shrug's decentralized ethos.

6. Conclusion

Shrug introduces a novel approach to memecoin engagement by combining token purchases with a rewarding SHRUG token distribution model. By integrating with WoofSwap and prioritizing security, Shrug offers a transparent and user-friendly platform for memecoin enthusiasts. The project is designed to foster participation in the memecoin ecosystem while maintaining fairness and decentralization.
